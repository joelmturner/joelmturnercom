## Hooks

```typescript
// add type before parens
React.useState<boolean>(false);
React.useRef<HTMLDivElement>(null);
React.useCallback(
  function () {
    if (theMoon) {
      setSomethingAwesome();
    }
  },
  [theMoon],
);
```
