# Uses

Here are some of the equipment, services, and software that use most frequently in my web development. _Disclosure - Some of the links may be affiliate links_.

## Hardware

15" Macbook Pro (2017) - The main machine I use.

27" iMac (mid 2010) - Secondary/backup machine.

[Sony WH-1000XM3](https://www.sony.com/electronics/headband-headphones/wh-1000xm3) - Great quality, noise cancelling headphones.

## Project/Task Management

[Todoist](https://todoist.com/r/joelmturner_rooqqg) - My goto todo app.

[Glo Boards](https://www.gitkraken.com/boards?referralCode=78cNgZYg) - Excellent Kanban style boards for useful tracking of projects.

## Development

[GitKraken Pro](https://www.gitkraken.com/invite/78cNgZYg) - My favorite GUI for git. Good features, great design, fun changelogs. You can access the Glo Boards directly in the in client as well.

### Editor and Terminal

[VS Code](https://code.visualstudio.com/) - My current preferred code editor.

- Theme - [Night Owl](https://marketplace.visualstudio.com/items?itemName=sdras.night-owl) from Sarah Drasner
- Extensions
  - [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker) - this has helped me fix so many typo caused bugs
  - [ESlint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) - code linting
  - [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) - great for collaboration and checking when something was written
  - [Peacock](https://marketplace.visualstudio.com/items?itemName=johnpapa.vscode-peacock) - extra theme goodies for VS Code
  - [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) - code formatting
  - [Project Manager](https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager) - easily manage and switch repos
  - [Todo Tree](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree) - a great way to find all the todos that are left in the code
  - [Settings Sync](https://marketplace.visualstudio.com/items?itemName=Shan.code-settings-sync) - helps keep VS Code settings in sync between computers
- Font - [Fira Code](https://github.com/tonsky/FiraCode) - excellent free font with ligatures

[iTerm2](https://www.iterm2.com/) - My favorite terminal alternative.

- ZSH and [Oh My ZSH](https://ohmyz.sh/) - Helpful goodies that can be used with iTerm2.
- Theme - [Night Owl](https://github.com/nickcernis/iterm2-night-owl) - Adaptation of Sarah Drasner's theme.

### Browsers

[Chrome](https://www.google.com/chrome/) - Currently, my main browser.

- Extensions
  - [OneTab](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=en) - clean up and manage tabs
  - [1Password X](https://chrome.google.com/webstore/detail/1password-x-%E2%80%93-password-ma/aeblfdkhhhdcdjpifhhbdiojplfjncoa?hl=en) - extension for the best password manager
  - [Awesome Screenshot & Screen Recorder](https://chrome.google.com/webstore/detail/awesome-screenshot-screen/nlipoenfbbikpbjkfpfillcgkoblgpmj) - a great screenshot tool for web pages
  - [Grammarly for Chrome](https://chrome.google.com/webstore/detail/grammarly-for-chrome/kbfnbcaeplbcioakkpcpgfkobkghlhen) - great for writing emails/papers

[Firefox](https://www.mozilla.org/en-US/firefox/) - My second-most used browser.

- Extensions
  - [1Password](https://1password.com/browsers/firefox/) - extension for the best password manager

## Desktop Apps

[Alfred](https://www.alfredapp.com/) - Productivity app. One of my most used apps and the first one I install on a new computer. Here's a little about [how I use Alfred](/blog/productivity-practices-front-end-development/#app-launching).

[1Password](https://1password.com/) - The best password manager.

[Local](https://localwp.com/) - The best way I've found for developing on WordPress sites locally.

## Hosting

[Netlify](https://www.netlify.com/) - My favorite way to host JamStack projects.
[SiteGround](https://www.siteground.com/go/inclinedcreative) - I mainly use this for WordPress sites.

## Learning/Reading

[Feedly](https://feedly.com/) - Solid RSS reader.

[Pocket](https://app.getpocket.com/) - Save/read pages on the web.

[Dev.to](https://dev.to/) - Great tech community and articles.

[PocketCast](https://www.pocketcasts.com/) - My favorite podcast player.
